## This bot was intentionally made as a portfolio bot to showcase ability to use APIs, and overall ability of discord.py

## STRUCTURE
- main.py is the program where the bot is launched, and where cogs are loaded.
- /cogs is the directory of cogs
- __init__.py is empty, it's there to show that cogs is a package
- weather.py program has weather related commands
- moderation.py program has moderation related programs
- utility.py program has utility related programs
- setup.py program has AutoMod implementation, and setup ease
